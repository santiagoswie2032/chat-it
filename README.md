Chat-It

A simple real-time chat application built with a React frontend and a Node.js backend. Users can join chat rooms and send messages instantly with a clean and minimal interface.

Live Demo:
https://chat-it-qm2l.onrender.com/

Features ->

Real-time messaging

Join chat rooms

Simple and responsive UI

Instant message updates using WebSockets

Lightweight and easy to use
